08. Т.к. у нас файл ~/.ssh/authorized_keys отсутствует, необходимо его создать и поместить туда какой-либо публичный rsa-ключ.
Для начала сгенерируем его: ssh-keygen -t rsa, где "-t rsa" задаст тип ключа - RSA.
При помощи команды scp (см. man scp) копируем содержимое ~/.ssh/id_rsa.pub (публичного ключа) в файл ~/.ssh/authorized_keys:
scp ~/.ssh/id_rsa.pub mwilbur@127.0.0.1:/home/mwilbur/.ssh/authorized_keys
Done, наш файл существует и хранит в себе публичный ключ, сгенерированный ранее.

Источник: https://www.dmosk.ru/miniinstruktions.php?mini=scp-without-password
